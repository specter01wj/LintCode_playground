# Group Anagrams

> Given an array of strings, group anagrams together.

## Example

Given: `["eat", "tea", "tan", "ate", "nat", "bat"]`, 

Return: 

```
[
  ["ate", "eat","tea"],
  ["nat","tan"],
  ["bat"]
]
```

## Notice

All inputs will be in lower-case.
